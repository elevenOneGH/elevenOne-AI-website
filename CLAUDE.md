# Websiteerstellung

## Tech Stack
- **Framework:** Next.js 15  
- **Sprache:** TypeScript  
- **Styling:** Tailwind CSS  
- **Animationen:** Framer Motion  

## Design-Regeln

### Nutzeranforderungen klären
- Verwende das **AskUserQuestion Tool**, um gezielte Fragen zum gewünschten Websitedesign zu stellen.
- Ziel: Die visuelle und funktionale Vorstellung des Nutzers möglichst exakt erfassen und umsetzen.

### UI / UX Entscheidungen
- Nutze den **frontend-design Skill** für alle UI- und Layoutentscheidungen.
- Nutze **UI/UX Pro Max** zur Generierung eines konsistenten **Design-Systems** (Farben, Typografie, Spacing, Komponenten).

### Komponenten-Inspiration
- Verwende bei Bedarf **21st.dev** als Inspirationsquelle für moderne UI-Komponenten (nur wenn explizit sinnvoll oder vorgegeben).

## Design-Prinzipien
- **Keine generischen AI-Aesthetics**
- **Mutige, unverwechselbare Designentscheidungen**
- **Klare visuelle Hierarchie**
- **Modernes, hochwertiges UI**

## Performance
- Optimierung für **Core Web Vitals**
- Minimale Bundle-Größe
- Effiziente Nutzung von Next.js Features (z. B. Server Components, Image Optimization)
